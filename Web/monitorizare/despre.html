IoT și sănătatea

Tehnologia IoT permite monitorizarea continuă a parametrilor vitali, cum ar fi temperatura și pulsul. Aceste date pot fi analizate în timp real, oferind informații valoroase pentru prevenirea problemelor de sănătate.

Balbismul și impactul asupra sănătății

Balbismul este o tulburare de vorbire adesea corelată cu stresul și anxietatea. Aceste stări influențează parametri fiziologici precum ritmul cardiac și temperatura corporală.

  • Ritmul cardiac crește în timpul crizelor de anxietate.
  • Temperatura poate fluctua în situații de stres intens.
  • Monitorizarea acestor valori ajută la înțelegerea și gestionarea balbismului.

Scopul proiectului

Proiectul nostru propune un sistem IoT simplu, dar eficient, care măsoară temperatura și pulsul, stochează valorile pe un server și le afișează într-o interfață web ușor de accesat.

Tehnologii utilizate

  • ESP32 + senzor DHT22 + senzor puls
  • Interfață web HTML/CSS + Chart.js
  • Stocare locală în MySQL prin PHP

Beneficii

  • Monitorizare în timp real
  • Feedback instant asupra stării fiziologice
  • Acces de oriunde din rețeaua locală
